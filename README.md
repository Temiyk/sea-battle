# 🚢 Sea Battle Online

Современная реализация классической настольной игры «Морской Бой» с возможностью игры по сети в реальном времени. Проект построен на стеке Node.js + Socket.io, что обеспечивает мгновенный обмен данными между игроками.

## ✨ Особенности

**Сетевая игра:** Создавайте комнаты и приглашайте друзей по ID комнаты.

**Гибкая расстановка:** Вы можете расставить корабли вручную кликами по полю или положиться на рандом, нажав кнопку «Случайно».

**Интуитивный интерфейс:** Современный дизайн с анимациями и красками.

**Мониторинг флота:** Специальные миниатюры рядом с полями показывают, сколько кораблей осталось у вас и у противника.

**Умная логика:** Автоматическое закрашивание клеток вокруг потопленного корабля, соблюдение правил расстановки и невозможность неправильного хода.

**Реванш:** Возможность начать новую партию сразу после окончания текущей с перестановкой кораблей и без переподключения.

## 🛠 Технологии

**Backend:** Node.js, Express

**Real-time:** Socket.io

**Frontend:** HTML5, CSS3 (Modern UI), Native JavaScript
